### 题目描述

以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 *一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间* 。

### 知识点

1. **intervals.sort( (a, b) => a-b )**

   - 遍历数组并对元素两两比较
   - 根据比较函数的结果决定元素顺序
   - 原地修改数组（返回排序后的数组，但原数组也会改变

   ![76493895898](C:\Users\ADMINI~1\AppData\Local\Temp\1764938958989.png)

   ### 答案

```js
var merge = function (intervals) {
  //特殊情况：没有元素或者只有一个元素时 直接返回
  if(intervals === null || intervals.length ===0) return intervals  // 重新从小到大排序
  intervals.sort((a,b) => a[0] - b[0])
  let result = [intervals[0]]   
  for(let i = 1; i < intervals.length; i++) {
    let current = intervals[i]
    let last = result[result.length - 1]    // 当前区间和结果中存放的有重叠 => 修改结果中存放的值
    if(current[0] <= last[1]) last[1] = Math.max(current[1],last[1])
    else result.push(current) 
  }
  return result
};
```

###   思路

result 表示结果数组，初始为第一个区间，current 表示当前的区间，last表示result中最后一个数组

开启遍历，判断 last[1] 和 curtent[0]，合并 => 更新到 result
								不能合并 => 将 current 区间推入 result 数组



比如 intervals = [ [2, 6], [1, 3], [8, 10], [15, 18]]

1.首先要排序 => intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]

2.令result=[ [1,3] ]

3.遍历intervals：intervals[i]中的第一个元素 与 result中最后一个元素中的第一个元素比较 => [2,6] 和 [1,3]比较 => 2小于3 => 要合并 =>令[1,3]中的3为max(6,3) =>result=[ [1,6] ]

## 128.最长连续序列

### 题目描述

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

### 答案

```js
var longestConsecutive = function (nums) {
  if(nums.length === 0) return 0
  nums.sort((a, b) => a - b) // 先从小到大排序
  nums.push(1000) // 随便添加一个数
  let temp = nums[0]
  let arr = []
  let result = 1
  // 找到去掉重复数字的数组
  for (let i = 1; i < nums.length; i++) {
    if (nums[i] !== temp) {
      arr.push(temp)
      temp = nums[i]
    }
  }
  // 找到连续数字的个数即为答案
  let sum = 1
  let temp1 = arr[0]
  for (let i = 1; i < arr.length; i++) {
    if (temp1 + 1 === arr[i]) {
      sum++
      temp1++
    }
   else {
      result = Math.max(result, sum)
      temp1 = arr[i]
      sum = 1
    }
  }
  return Math.max(result, sum)
};
```

## 11.盛最多水的容器

### 题目描述

给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

### 答案

- 思路：用双指针i，j循环height数，i，j对应高度较小的那个先向内移动，不断计算面积，更新最大面积

```js
var maxArea = function (height) {
    let max = 0 // 返回值
    let square = 0 // 所盛水量 
    let j = height.length - 1 // 右指针
    for (let i = 0; i < height.length ; i++) {
      square = Math.min(height[i], height[j]) * (j - i ) 
      max = Math.max(max,square)
      // 左端比右端大的时候，右指针向左移，左指针应该不变
      if(height[i] > height[j]){
        j--
        i-- //这里-1是为了弥补for循环中的+1 => 左指针不动
      }
    }
    return max
  };
```

### 答案

```js
var firstMissingPositive = function (nums) {
  nums.sort((a,b) => a-b) // 从小到大排列
  if(!nums.includes(1)) return 1 // 没有1
  else {
    const arr = nums.filter(item => item >= 1)   
    for(let i = 0; i < arr.length; i++) {
      if(arr[i] + 1 !== arr[i+1] && arr[i] !== arr[i+1]) return arr[i] + 1
    }
  }   
};
```

先判断数组中有没有1

有1 => 找到大于1的数组 遍历，

​	    当后一个数不等于前一个数+1 并且 后一个数不等于前一个数的时候，当前数+1就是缺失的正数

## 238.除自身以外数组的乘积

### 题目

给你一个整数数组 `nums`，返回 数组 `answer` ，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积 。

题目数据 **保证** 数组 `nums`之中任意元素的全部前缀元素和后缀的乘积都在  **32 位** 整数范围内。

请 **不要使用除法，**且在 `O(n)` 时间复杂度内完成此题。

### 答案

```js
 var productExceptSelf = function (nums) {
   const n = nums.length
   const re = []
   const pre = []
   pre[0] = 1
   const sub = []
   sub[n-1] = 1
   // 算左侧除自身外的乘积
   for(let i = 1; i < n; i++) {
     pre[i] = pre[i-1] * nums[i-1] // 左侧当前数 = 左侧前一个数 * 前一个数
   }
   // 算右侧除自身外的乘积
   for(let j = n-2; j >= 0; j--) {
     sub[j] = sub[j+1] * nums[j+1] // 右侧当前数 = 右侧后一个数 * 后一个数
   }

   for(let k = 0; k < n; k++ ){
     re[k] = pre[k] * sub[k] // 对应结果 = 对应左 * 对应右
   }
     
   return re
 };
```

answer[i] 等于 nums 中除了 nums[i] 之外其余各元素的乘积。换句话说，如果知道了 i 左边所有数的乘积，以及 i 右边所有数的乘积，就可以算出 answer[i]。

于是：

定义 pre[i] 表示从 nums[0] 到 nums[i−1] 的乘积。
定义 suf[i] 表示从 nums[i+1] 到 nums[n−1] 的乘积。
我们可以先计算出从 nums[0] 到 nums[i−2] 的乘积 pre[i−1]，再乘上 nums[i−1]，就得到了 pre[i]，即

pre[i]=pre[i−1]⋅nums[i−1]
同理有

suf[i]=suf[i+1]⋅nums[i+1]
初始值：pre[0]=suf[n−1]=1。按照上文的定义，pre[0] 和 suf[n−1] 都是空子数组的元素乘积，我们规定这是 1，因为 1 乘以任何数 x 都等于 x，这样可以方便递推计算 pre[1]，suf[n−2] 等。

算出 pre 数组和 suf 数组后，有

answer[i]=pre[i]⋅suf[i]

## 43.接雨水

### 题目

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**示例 1：**

![77487223151](C:\Users\ADMINI~1\AppData\Local\Temp\1774872231514.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
```

### 答案

```js
var trap = function (height) {
  let L = 0; //左指针
  let R = height.length - 1; //右指针
  let L_max = 0; //左边最大值
  let R_max = 0; //右边最大值
  let result = 0;

  while (L <= R) {
    L_max = Math.max(L_max, height[L]);
    R_max = Math.max(R_max, height[R]);
    //左边最大值 < 右边最大值：下标 L 处能接的雨水量等于 L_max - height[L]
    if (L_max < R_max) {
      result = result + (L_max - height[L]);
      L++;
    } else {
      result = result + (R_max - height[R]);
      R--;
    }
  }
  return result;
};
```

**核心思想**：**对于每个位置，能接的雨水量 = min(左边最高柱子, 右边最高柱子) - 当前高度**

- 使用左右指针从两端向中间移动

- 维护左右两侧的**当前最大值**

- 哪边最大值小，就计算哪边的雨水量

## 189.轮转数组

### 题目

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

**示例 1:**

```
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
    向右轮转 1 步: [7,1,2,3,4,5,6]
    向右轮转 2 步: [6,7,1,2,3,4,5]
    向右轮转 3 步: [5,6,7,1,2,3,4]
```

### 答案

```js
var rotate = function (nums, k) {
  let n = nums.length;
  let newArray = [];

  for (let i = 0; i < nums.length; i++) {
    newArray[(i + k) % n] = nums[i];
  }

  for (let j = 0; j < nums.length; j++) {
    nums[j] = newArray[j];
  }

  return nums;
};
```

使用额外的数组来将每个元素放至正确的位置。用 n 表示数组的长度，我们遍历原数组，将原数组下标为 i 的元素放至新数组下标为 (i+k)modn 的位置，最后将新数组拷贝至原数组即可。

```js
var climbStairs = function (n) {
  let p = 0;
  let q = 0;
  let r = 1;
  for (let i = 1; i <= n; i++) {
    p = q;
    q = r;
    r = p + q;
  }
  return r;
};
```
